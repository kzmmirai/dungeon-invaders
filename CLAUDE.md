# DUNGEON INVADERS - プロジェクト概要

## ゲームコンセプト
ダンジョン探索RPG × インベーダー風シューティング戦闘のレトロドット絵ゲーム。
ブラウザ上でローカル動作（HTML5 Canvas + vanilla JS、外部ライブラリなし）。

## ゲームフロー
タイトル(NEW GAME/CONTINUE) → ダンジョン探索(1体の敵を見つける) → インベーダー戦闘 → 勝利で階段出現 → 次ステージ → STAGE5ボス撃破でクリア

## 描画方式
- Canvas解像度: 240×160px → CSSで画面いっぱいに拡大（アスペクト比維持）
- `image-rendering: pixelated` でレトロドット感
- タイトル/ゲームオーバー/クリア画面: HTML DOMオーバーレイ（Press Start 2Pフォント）
- アイテムスワップモーダル: HTML DOM（Canvas外）

## ファイル構成
```
index.html          - エントリーポイント、Canvas + DOMオーバーレイ
css/style.css       - ダークテーマ、レトロUI、DOMオーバーレイスタイル
js/constants.js     - 定数、カラーパレット、バランス数値、時間制限
js/sprites.js       - ドット絵データ定義・描画関数（絵の修正はここ）
js/player.js        - プレイヤー（ステータス、セーブ/ロード、ダメージ計算）
js/enemies.js       - 敵データ定義、ランダム散布配置、ボス+手下20体
js/items.js         - アイテム定義、ドロップテーブル（5階層分）
js/dungeon.js       - ランダムマップ生成、フロア別テーマ色、ミニマップ、松明グロウ
js/battle.js        - インベーダー風戦闘（自由移動、弾数制限、時間制限、ピックアップ）
js/ui.js            - HUD、SE（Web Audio）、BGM（パターン生成）、エフェクト
js/main.js          - シーン管理、セーブ統合、アイテムスワップキュー
```

## ステージ構成（全5ステージ、各階に敵1体）
- STAGE 1 地下洞窟(青): スライム12体 / 25秒
- STAGE 2 暗黒の森(緑): コウモリ+スライム15体 / 30秒
- STAGE 3 骨の迷宮(茶): スケルトン+コウモリ16体 / 35秒
- STAGE 4 魔王の城(紫): ダークメイジ+ゴーレム18体 / 40秒
- STAGE 5 ドラゴンの巣(赤): ドラゴン(3フェーズボス)+手下20体 / 70秒

## 戦闘システム
- 自機: 上下左右自由移動（画面全域）
- 弾数制限あり（ステージごとに25-35発）
- 弾補給・回復ハートが画面内を縦横無尽に飛び回る（早めに取らないと消える）
- 敵は個別にランダムな速度・方向で動く（0.2x〜2.0x速度差）
- ボスは3フェーズ制（見た目が崩壊、フェーズごとに弾幕激化+手下召喚）
- 撃破演出: パーティクル爆発+リングエフェクト+スローモ+画面シェイク

## 音響
- BGM: Web Audio APIでパターン生成（ダンジョン/バトル/ボス戦）
- SE: 射撃、被弾、敵撃破（爆発ノイズ）、ボス撃破（連鎖爆発）、回復（チャイム）、レベルアップ（アルペジオ）、弾補給、時間警告、ゲームオーバー、勝利ファンファーレ

## セーブ機能
- localStorage使用、Qキーで手動セーブ、階段で自動セーブ
- タイトル画面でCONTINUE選択可能

## 操作
- 探索: 矢印キー移動、1-5=アイテム使用、Q=セーブ
- 戦闘: ←→↑↓=自由移動、Space=弾発射、1-5=アイテム使用

## 技術方針
- ES Modules、ビルド不要
- `python3 -m http.server 8080` → localhost:8080 でプレイ
- ドット絵修正は sprites.js のみ
- `/check` コマンドで構文・ブレース・import/export・HTML参照を一括チェック
